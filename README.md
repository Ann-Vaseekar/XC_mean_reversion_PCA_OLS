# Cross-Sectional Crypto Mean Reversion Strategy (PCA vs OLS)

In this project, we investigated PCA vs OLS residual modelling for a cross-sectional mean reversion strategy across 70 crypto assets (4-hour data, 2022-2026), implementing rank-threshold signal transformation, dollar-neutral portfolio construction, and grid-based hyperparameter optimisation.

Findings:
•	PCA model delivered superior out-of-sample performance (net Sharpe 1.07 vs 0.33 for OLS), validated through information coefficient decay analysis, drawdown decomposition, and transaction-cost sensitivity testing.
